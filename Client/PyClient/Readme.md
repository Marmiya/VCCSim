# PyClient of VCCSim

## Prerequisites

Run the setup script

    ``` powershell
    .\setup.cmd
    ```

## Usage

The directory structure should be like this:

    ``` file
    PyClient
    ├── Readme.md
    ├── setup.cmd
    ├── VCCSim
    │   ├── __init__.py
    │   ├── VCCSimClient.py
    │   ├── VCCSim_pb2_grpc.py
    │   └── VCCSim_pb2.py
    └── test_scripts
    ```
